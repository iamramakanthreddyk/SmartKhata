### Component Patterns

**Buttons**:
- Primary (filled): Full width at bottom
- Secondary (outlined): For destructive actions
- Tertiary (text): For navigation

**Input Fields**:
- Labels above fields
- Error text in red below
- Cursor auto-focuses for data entry

**Cards**:
- Rounded corners (8px)
- Subtle shadow on tap
- Swipe-to-delete enabled
